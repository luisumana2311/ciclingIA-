# IA Cycling

AI-powered web application designed to support cyclists with training planning, athlete profile management and adaptive workout recommendations.

## Overview

IA Cycling is a full-stack project focused on creating a digital training assistant for cyclists. The system is designed to generate training plans based on athlete information, available time, goals and training context.

## Tech Stack

- JavaScript
- Node.js
- Express.js
- MongoDB
- React
- CSS

## Main Features

- Athlete profile management
- Training plan generation
- Weekly training structure
- Fatigue and availability-based logic
- Backend API for cycling training data
- Full-stack project architecture

## Project Purpose

This project was created as a personal software development project combining backend development, full-stack architecture and cycling performance logic.

## Future Improvements

- Garmin or Strava integration
- More advanced AI recommendations
- User authentication
- Training history dashboard
- Deployment to production
